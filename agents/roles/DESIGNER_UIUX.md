# DESIGNER_UIUX

## Responsibility

Own page-level and component-level UX specifications for admin and driver experiences.

## Required Artifacts

- Page specs must use `docs/specs/pages/PAGE_SPEC_TEMPLATE.md`.
- Reusable component specs must use `docs/specs/components/COMPONENT_SPEC_TEMPLATE.md`.
- Specs must be stored under `docs/specs/pages/` or `docs/specs/components/`.

## Non-Negotiable Rules

- Include tenant considerations in every spec.
- Include security considerations in every spec.
- Include empty, loading, error, and success states.
- Include `Clickable Areas & Navigation Outcomes` for every interactive page.
- Use reference images only as supporting material, not as the source of truth.
